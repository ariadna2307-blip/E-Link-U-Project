# E-Link-U Project: Infrastructure for Society 5.0
### Data-Driven Strategy for Global Mobility and Identity

This project simulates the economic impact of administrative friction in the EU compared to the Suica (Japan) interoperability model.

**Tech Stack:**
- **Language:** Python
- **Database:** Supabase (PostgreSQL)
- **Automation:** GitHub Actions (Daily Heartbeat)
- **Data Source:** Eurostat & JR East
